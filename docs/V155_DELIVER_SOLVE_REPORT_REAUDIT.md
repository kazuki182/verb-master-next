# Ver.155 DELIVER / SOLVE / REPORT Reaudit

## 対象
- DELIVER
- SOLVE
- REPORT

## 監査結果
- DELIVER: 句動詞6個
- SOLVE: 句動詞4個
- REPORT: 句動詞5個
- collocations は3動詞とも空
- 基本例文の赤文字は動詞のみ
- 句動詞例文の赤文字は動詞 + 前置詞/副詞まで
- 目的語は赤文字対象外
- 基本と句動詞の重複を整理
- solve before を除外
- solve the root cause の不自然な表現を修正
- report back と report back to の重複を整理

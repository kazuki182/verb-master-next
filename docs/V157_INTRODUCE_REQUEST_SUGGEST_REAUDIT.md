# Ver.157 INTRODUCE / REQUEST / SUGGEST Reaudit

## 対象
- INTRODUCE
- REQUEST
- SUGGEST

## 監査結果
- INTRODUCE: 句動詞5個
- REQUEST: 句動詞4個
- SUGGEST: 句動詞3個
- collocations は3動詞とも空
- 基本例文の赤文字は動詞のみ
- 句動詞例文の赤文字は動詞 + 前置詞/副詞まで
- 目的語は赤文字対象外
- 基本と句動詞の重複を整理
- be introduced to は introduce to に統合
- introduce gradually / newly introduced を句動詞から除外
- request for / upon request / request in writing を句動詞から除外
- request someone to do を request that someone do に修正
- suggest that / suggest doing は基本側へ整理

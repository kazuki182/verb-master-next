# Verb Master Ver.137 動詞検索BOX 設計書

## 目的

既存の検索BOXを、単純な文字列検索から「動詞・日本語訳・句動詞・例文を横断して探せる検索」に強化する。
スマホでの学習中に、ユーザーが「会議」「承認」「find out」「句動詞」などから必要な動詞へ早く到達できることを目的にする。

## 実装方針

### 1. 検索対象

- 動詞名
- id
- カナ
- IPA
- コアイメージ
- coreDetail
- 基本のタイトル
- 基本のpattern
- 基本の解説
- 基本例文の英語・日本語
- 句動詞のphrase
- 句動詞の日本語訳
- 句動詞のpoint

### 2. 検索方式

- NFKC正規化
- 小文字化
- ハイフン、スラッシュ、アンダースコアをスペース扱い
- 複数語はスペース区切りのAND検索
- 例：`顧客 連絡` は両方を含む動詞だけ表示

### 3. フィルター

- 全範囲
- 基本だけ
- 句動詞だけ
- 例文だけ

### 4. 表示範囲

- 全動詞
- 解放済みだけ
- 未解放だけ

### 5. 並び順

- 番号順
- 一致度順

一致度順は、動詞名・idの一致を最優先し、句動詞・基本・例文の一致を加点する。

### 6. スマホ操作

- 推奨キーワードチップを追加
- 検索条件リセットを追加
- 表示件数と検索条件を常時表示
- 一致した領域を「基本で一致」「句動詞で一致」「例文で一致」として表示

## データ保存への影響

検索BOXは表示ロジックのみのため、Supabase保存基盤には影響しない。
プロフィール画像、ニックネーム、目標日、学習データの保存仕様はVer.136を維持する。

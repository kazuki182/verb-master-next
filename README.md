- SEE
- LOOK

## 現在の作り直し済み動詞

## 重要ルール
- 画面上のカテゴリは「基本」と「句動詞」のみです。
- 熟語、コロケーション、文型、表現、よく使うフレーズは別カテゴリとして出しません。
- 必要な内容は「基本」に含めます。
- 句動詞は自然なものだけ入れ、無理に10個へ増やしません。

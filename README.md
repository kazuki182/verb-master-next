# Verb Master

Ver.117 WATCH / HEAR / LISTEN 追加版です。

## 今回の対象動詞
- WATCH
- HEAR
- LISTEN

## 現在の作り直し済み動詞
GET / TAKE / MAKE / GIVE / HAVE / GO / COME / PUT / KEEP / FIND / SEE / LOOK / WATCH / HEAR / LISTEN

## 重要ルール
- 画面上のカテゴリは「基本」と「句動詞」のみです。
- 熟語、コロケーション、文型、表現、よく使うフレーズは別カテゴリとして出しません。
- 必要な内容は「基本」に含めます。
- 句動詞は自然なものだけ入れ、無理に10個へ増やしません。

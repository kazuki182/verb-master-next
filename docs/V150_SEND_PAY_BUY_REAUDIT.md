# V150 SEND / PAY / BUY Reaudit

## 対象
- SEND
- PAY
- BUY

## 方針
- カテゴリは「基本」「句動詞」のみ。
- collocations は空に整理。
- 基本の赤文字は動詞のみ。
- 句動詞の赤文字は動詞＋前置詞・副詞まで。
- 目的語は赤文字対象外。
- Ver.145 のログイン・復元・保存基盤を維持。
- Ver.146 のユーザー画面クリーンアップを維持。
- Ver.140 のシンプル頭文字検索を維持。

## 句動詞数
- SEND: 9
- PAY: 7
- BUY: 5

## 備考
SEND は送付・発送・転送を中心に整理。
PAY は支払い・返済・報われる表現を整理し、pay attention to は基本側で扱う。
BUY は buy off を除外し、自然な5個に絞った。

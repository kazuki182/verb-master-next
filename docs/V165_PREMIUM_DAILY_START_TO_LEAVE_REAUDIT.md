# Ver.165 Premium Daily Examples Re-audit: START to LEAVE

## Scope

Premium-only daily conversation examples were re-audited for these ten verbs:

- START
- STOP
- TRY
- HELP
- SHOW
- TELL
- ASK
- CALL
- RUN
- LEAVE

## Result

- Premium audit progress: 30 / 124 verbs
- Daily examples added or revised in this batch: 226
- Every retained basic meaning and phrasal-expression block has exactly two daily examples.
- Complete English-sentence duplicates were removed within this batch.
- Focus text was checked against each English sentence.
- Basic examples emphasize only the target verb form.
- Phrasal-expression examples emphasize the verb plus required particle/preposition without including the object.

## Expression cleanup

- STOP: removed duplicate `stop from` block.
- TRY: removed low-priority `try back` block.
- HELP: removed overlapping `help out with` and low-priority `help along` blocks.
- SHOW: removed unnatural or overlapping `show into` and `show over` blocks.
- TELL: removed duplicate `tell about` block.
- ASK: replaced `ask after` with everyday `ask out`.
- RUN: added the essential physical meaning “to run”.
- LEAVE: replaced overlapping `leave on` / `leave in` blocks with `leave up to` / `leave with`.

Premium段階課金・無料3動詞設計版

反映内容
- 無料範囲を3動詞に設定
- 第1課金で30動詞まで解放（追加4〜30、残り27動詞）
- 第2課金で60動詞まで解放（追加31〜60）
- 第3課金で90動詞まで解放（追加61〜90）
- 第4課金で120動詞まで解放（追加91〜120）
- 各パックは500円想定、累計金額を30=500円 / 60=1,000円 / 90=1,500円 / 120=2,000円に整理
- Upgrade / Checkout / Complete / Purchases の表示文言を段階課金に統一
- Stripe Price ID名を NEXT_PUBLIC_STRIPE_PRICE_30 / 60 / 90 / 120 に整理
- 管理者テスト操作の「次の段階を反映」を30→60→90→120の順に変更

確認ポイント
1. 無料ユーザーで1〜3番だけ開けるか
2. 4番以降が有料誘導になるか
3. Upgrade画面で30 / 60 / 90 / 120の段階表示になるか
4. 仮購入で30→60→90→120の順に解放されるか
5. 購入履歴の表示がStep表記になるか

# QUALITY_AUDIT_V89

## Ver.89: 動詞品質改善 第1弾

### 方針
- 124語を一気に直さず、例文が極端に少ない箇所から3語ずつ慎重に改善する。
- 確認順は「動詞の基本 → 熟語・よく使う表現 → 句動詞 → 日常例文」。
- 今回は日常例文の全面修正には入らず、教材の土台になる基本用法・熟語・句動詞の仕事例文を優先した。

### スキャン結果
Ver.88の全動詞データをスキャンし、examples が3件未満のブロックを抽出。
最も薄かった主要対象は以下。

1. RUN: 熟語3件・句動詞3件が各1例文のみ
2. START: 基本用法1種類のみ、熟語3件・句動詞2件が各1例文のみ
3. SHOW: 基本用法1種類のみ、熟語3件・句動詞2件が各1例文のみ

### 今回作り込んだ3語

#### START
- 基本用法: 1 → 6
- 熟語・よく使う表現: 3 → 5
- 句動詞: 2 → 5
- 各ブロックの仕事例文: 3件に統一
- 追加した主な型:
  - start + 名詞 / start + V-ing
  - start at + 時間
  - start with + 名詞
  - start from + 起点
  - start + system / machine
  - start as + 役割・形
  - start over / start off / start up / start out

#### SHOW
- 基本用法: 1 → 5
- 熟語・よく使う表現: 3 → 5
- 句動詞: 2 → 5
- 各ブロックの仕事例文: 3件に統一
- 追加した主な型:
  - show + 人 + 物
  - show + 結果・傾向・数値
  - show + 人 + how to do
  - show that + 文
  - show + 人 + 場所
  - show up / show around / show off / show through / show someone in

#### RUN
- 基本用法: 1 → 5
- 熟語・よく使う表現: 3 → 5
- 句動詞: 3 → 5
- 各ブロックの仕事例文: 3件に統一
- 追加した主な型:
  - run + meeting / business / project
  - run + test / system / check
  - run until / run from / run for
  - cable/pipe/road + run
  - run late / run short / run smoothly
  - run into / run out of / run by / run through / run across

### 今回あえて未対応にしたもの
- Premium日常例文の全面修正
- get angry などの不自然な日常例文修正
- コアイメージのPremium折りたたみ化

理由: ユーザー方針に従い、日常例文は「動詞の基本 → 熟語 → 句動詞」が終わってから対応するため。
